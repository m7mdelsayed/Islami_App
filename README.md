# islami_app

Flutter application for reading Quran and Hadith build by using flutter basics widgets & provider state management & api for radio screen & light theme & dark theme & localization

## Screens it contains : 
=> Splash Screen                                                                                                                                                       
=> Quran Screen                                                                                                                                                         
=> Hadith Screen                                                                                                                                                       
=> Tasbeeh Screen                                                                                                                                                       
=> Radio Screen                                                                                                                                                         
=> Settings Screen                                                                                                                                                     
=> Sura Details Screen                                                                                                                                                 
=> Hadith Details Screen
